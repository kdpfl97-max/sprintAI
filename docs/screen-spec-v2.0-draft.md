# Screen Spec v2.0 Draft

## 대시보드 PM
- 핵심 지표: 완료율, 진행 중, 남은 기간, 확인 필요
- 확인 필요 항목: 미배정, 블로커, 마감 임박, 과부하
- 팀원별 현황: 잔여 시간, 완료 시간, 펼침 태스크 목록
- 간트: 전체 태스크 일정과 오늘 위치

## 대시보드 팀원
- 내 배정
- 내 완료율
- 지금 할 일
- 다음 할 일
- 검토 요청 중
- 내가 기다리는 업무 / 나를 기다리는 업무
