# SprintAI v2.0 Obsidian Summary

## 한 줄 정의
SprintAI는 팀의 계획과 실행 사이에서 사라지는 정보를 구조화하는 가벼운 스프린트 도구다. AI는 초안 작성 보조다.

## 핵심 결정
- PM은 계획 확정, 위험 확인, 팀 조율을 한다.
- 팀원은 내 할 일, 마감, 블로커만 빠르게 확인한다.
- 부트캠프 팀 자체보다 강사/교육기관의 반복 사용을 본다.

## P0
- 할 일 캡처
- 계획 확정
- 실행 중 확인 필요 항목 표시

## 보류
- Notion/Sheets 가져오기
- 외부 알림 자동화 고도화
- 조직 결제/관리자 기능
